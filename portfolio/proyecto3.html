API REST

Esta API rest, no es mas que una potente y muy confiable interfaz que tiene la capacidad de comunicar a cualquier sistema externo con una base de datos MySQL de una manera rapida y segura. Este proyecto, Forma parte de un sistema integral de domotica que tiene como objetivo principal el control y automatizacion de dispositovos de aires acondicionados de la mano de Sync-Air

Las tecnologias principales usadas para su desarrollo fueron Java con Spring-Boot y como bbdd usa un motor MySQL.

Tecnologías Utilizadas

  • Spring-Boot
  • MySQL
