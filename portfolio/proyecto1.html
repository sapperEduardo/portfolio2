Agenda Electrónica

Este proyecto fue desarrollado usando Java con su librería gráfica Swing y SQLite para el sistema de persistencia. Es una aplicación de escritorio que permite gestionar contactos de forma simple y rápida.

Ofrece funcionalidades como agregar, editar y eliminar contactos, todo almacenado en una base de datos SQLite local.

Tecnologías Utilizadas

  • Java-Swing
  • SQLite
